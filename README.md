## Racial Bias Diagnostic Engine

**SOVRA-FCL-MHCE-v2.5© · FOXTROT-CHIMERA-LAMBDA · NFIE COMPLIANT©**

Sovra-FCL-LDA is a legal diagnostic tool that exposes racial bias in criminal case records through structural analysis. It ingests attorney case data, runs the CHIMERA contradiction diagnostic pipeline, and surfaces LDF-aligned counter-strategy indicators.

## Features

- **Excel Upload** — Accepts Sovra Legal Aid Framework .xlsx files. No backend required. Data never persists.
- **Search Engine Interface** — Search cases by ID, judge, charge type, jurisdiction, or any field.
- **CHIMERA Diagnostic Pipeline** — Three-axis contradiction analysis (Jury Bias, Sentence Disparity, Defense Adequacy)
- **LDF Strategy Indicators** — Auto-routes to Batson, Buck Pattern, Flowers Standard, or Sentencing challenge
- **Pattern Report** — Multi-case aggregate analysis for systemic evidence building
- **Sovra Voice** — NFIE-compliant field observation (no assertion, no persuasion)
- **Database Module** — Staged. Supabase architecture ready. Privacy review required before activation.

## Context Gates

- **Contradiction Analysis** — Surfaces C1/C2/C3 flags and CDI score per case
- **Narrative Drift** — Shows Punishment/Containment/Erasure/Redemption/Rehabilitation category
- **Zero-Sum Framing** — Detects zero-sum narrative structures in case context
- **LDF Strategy** — Surfaces applicable legal counter-strategy per case
- **Sovra Voice** — Generates field-level observational statement

## Sample Query

```
Search: JUDGE-001

  CDI scores, bias flags, and LDF strategy indicators
```
